# 一、初始化node项目
```
npm init -y
```


## server端

- 创建入口文件index.js


- 配置package.json

```json
"serve": "nodemon index.js"
```
- 下载nodemon

```shell
npm i -g nodemon
```



# 二、后台管理

- 安装elementUI

  ```shell
  vue add element
  ```

- 安装Router

  ```shell
  vue add router
  ```

- 安装axios

  ```shell
  npm i axios
  ```

  

# 三、服务端

```shell
npm i express@next mongoose cors
```
